Password Strength Checker

📘 Kenapa Password yang Kuat Itu Penting?

Password adalah garis pertahanan pertama dalam keamanan digital. Jika password mudah ditebak atau lemah, akun kamu bisa diretas hanya dalam hitungan detik.

⚠️ Risiko Menggunakan Password Lemah

  • Brute Force attack – Peretas mencoba ribuan kombinasi password secara otomatis.
  • Dictionary attack – Peretas menggunakan daftar password umum seperti “123456”, “admin”, atau “password”.
  • Credential stuffing – Jika kamu pakai password yang sama di banyak akun, peretas bisa mengakses semuanya hanya dengan satu kebocoran data.

    • ✅ Tips Membuat Password yang Kuat

  • Gunakan minimal 8 karakter (lebih panjang lebih aman).
  • Kombinasikan huruf besar, huruf kecil, angka, dan simbol.
  • Hindari nama, tanggal lahir, atau kata umum.
  • Jangan gunakan password yang sama untuk banyak akun.
  • Guanakan password manager untuk menyimpan password dengan aman.

    • 🔐 Apa Itu Password Manager?

    Password manager adalah aplikasi yang membantu kamu menyimpan dan menghasilkan password yang kuat dan unik untuk setiap akun, contohnya:

  • Bitwarden (gratis & open-source)
  • 1Password
  • LastPass
  • KeePassXC

    • 🧠 Tahukah Kamu?

  • 123456" adalah password yang paling sering digunakan di dunia — dan juga paling mudah diretas.
  • Kombinasi 12 karakter acak membutuhkan miliaran tahun untuk dibobol dengan brute-force, tergantung sistemnya.
